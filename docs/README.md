# desktop-web-demo 学习文档

这是一个学习项目。目标：参照 `multica` 的架构与开发方式，在本仓库复刻一套“web 与 Electron 桌面端同步开发”的前端，并做出 multica 风格的主要 UI。

---

## 文档组织

学习阶段文档按顺序阅读：

| 文档 | 主题 | 是否写代码 |
|---|---|---|
| `00-学习计划.md` | 总纲：目标、阶段路径、架构图 | 否 |
| `01-架构与思维模型.md` | 阶段 0：理论基础 | 否 |
| `02-monorepo-化.md` | 阶段 1：改造成 pnpm workspace | 是 |
| `03-抽取-packages-ui.md` | 阶段 2：抽共享 UI 包 | 是 |
| `04-抽取-packages-views.md` | 阶段 3：抽共享页面与布局 | 是 |
| `05-引入-electron.md` | 阶段 4：用官方脚手架初始化并接入 Electron 外壳 | 是 |
| `06-平台能力抽象.md` | 阶段 5：在官方 Electron 外壳上接入平台能力接口 | 是 |
| `07-复刻主要-ui.md` | 阶段 6：复刻 multica 风格主界面 | 待开始 |
| `08-打包与收尾.md` | 阶段 7：打包与工程收尾 | 待开始 |

参考文档：

| 文档 | 作用 |
|---|---|
| `design.md` | UI 设计规范，所有界面实现都以此为准 |
