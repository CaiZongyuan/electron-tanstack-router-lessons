Electron App 开启动画示意

用纯 HTML + Canvas 做了 4 个方向：收聚成形、发散转场、呼吸待机、完整启动流程。粒子坐标是程序模拟版，之后可以替换成你图标的精确 SVG 点位。

Version 01

旋涡收聚成形

适合开机第一段：背景先浮现，散点从外圈沿螺旋路径归位，中央球最后点亮。

Version 02

向外发散转场

适合 splash 结束：完整图标被激活后轻微向外释放，粒子淡出，主界面接入。

Version 03

静谧呼吸待机

适合加载等待：图标已经成形，点阵有轻微旋转和明暗波动，不会显得焦躁。

Version 04

完整启动流程

把 01 和 02 串起来：收聚成 logo，中心点亮，随后粒子轻微发散，露出主窗口。

说明：这版是动效「气质和节奏」示意，不是最终精确图标复刻。下一步可以把真实图标拆成 SVG，每个圆点用真实坐标驱动，效果会更贴近原图。
